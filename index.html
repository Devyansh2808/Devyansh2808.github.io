Featured

Ludere — Movie and TV Recommendations

I built this because every recommendation app I used felt like it forgot what I actually liked after a couple of sessions. Ludere uses TF-IDF across genres, keywords, cast, and overview text, then scores everything against a taste profile built from your likes and dislikes weighted by recency. New users get a Bayesian cold-start queue so there's no awkward empty state on day one. FastAPI backend, React 19 frontend with Zustand for state.

Automated Plant Watering Machine

Soil moisture sensor feeds into an Arduino that decides when to water automatically. Also built a manual override mode. The first version had 70% detection accuracy which wasn't good enough, so I spent time iterating on the calibration until it hit 92%. Logged over 1,000 readings and visualized them live in MATLAB.
